# WP WebBoot

A simple WordPress Webpack Bootstrap skeleton theme

## Features

+ WordPress skeleton theme
+ Webpack 4
+ Bootstrap 4
+ [wp-bootstrap-navwalker](https://github.com/wp-bootstrap/wp-bootstrap-navwalker)
